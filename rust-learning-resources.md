# Rust Learning Resources

## Official & Highly Recommended

### The Rust Book – Guessing Game

The official **_The Rust Programming Language_** book starts you off by building a simple guessing game in Chapter 2. It's a gentle introduction that explains concepts as you code. The book continues with more projects throughout.

### Rustlings

An interactive course of small exercises you complete in your local environment. You fix code to make tests pass, learning through doing. Great for building muscle memory with Rust syntax.

### Command Line Book

**_The Rust Programming Language: Command Line Applications in Rust_** walks you through building real CLI tools with proper error handling, file I/O, and testing.

## Web Development

_(Matches a frontend background)_

### Zero to Production in Rust

Builds a newsletter/email service from scratch. Covers web servers, databases, and deployment. This is a paid book but very comprehensive.

### Actix Web Tutorials

The Actix web framework provides official tutorials for building REST APIs and web services.

## Game Development

_(Matches a Unity background)_

### Bevy Engine Tutorials

Bevy is Rust’s most popular game engine. Its official examples are excellent hands-on tutorials. If you come from Unity, you’ll appreciate the entity-component-system (ECS) approach.

### Rust Sokoban Tutorial

A step-by-step tutorial that builds a complete Sokoban puzzle game.

---

## Recommendation

The official Rust book at **rust-lang.org** is genuinely excellent and free. A great starting point is the guessing game chapter, then branching out based on your interests.
